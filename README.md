# Simple Calculator

A basic calculator application created using HTML, CSS, and JavaScript.

## Features

- ✨ Clean and modern user interface
- 🧮 Basic arithmetic operations (addition, subtraction, multiplication, division)
- ⌨️ Keyboard support for numbers and operations
- 🎯 Clear and Delete functions
- 📱 Responsive design

## How to Use

1. Open `index.html` in your web browser
2. Click the number buttons to enter digits
3. Click operator buttons (+, -, *, /)
4. Click the equals button (=) to calculate the result
5. Use the Clear (C) button to reset
6. Use the Delete (DEL) button to remove the last digit

## Keyboard Shortcuts

- **Numbers**: 0-9
- **Operators**: +, -, *, /
- **Calculate**: Enter or =
- **Clear**: C
- **Delete**: Backspace

## Files

- `index.html` - HTML structure
- `style.css` - Styling and layout
- `script.js` - Calculator logic
